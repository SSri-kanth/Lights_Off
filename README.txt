

	----Functionalities of the application----

- User can see the popular movies and TV shows.
- User can also select the movie or TV show from the top rated section.
- User can search for Movies, TV Shows and Persons.


	----Libraries used----
- Android Card view, Constraint layout, recycler view, animated vectors, retrofit2
- SQLite database
- The Movie DataBase
- Broadcast reciever


** You need to get an API key from TMDB and paste it in the strings.xml**
